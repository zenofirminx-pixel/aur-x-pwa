Confidentialité

1. Stockage des données

Toutes vos conversations sont stockées localement sur votre appareil. ChatAurX n'envoie aucune donnée vers des serveurs externes.

Vos messages, historiques et préférences restent dans le localStorage de votre navigateur. Vous gardez le contrôle total.

2. Aucune collecte

  • Nous ne collectons pas vos conversations
  • Nous ne vendons aucune donnée personnelle
  • Nous ne trackons pas votre activité
  • Aucun cookie tiers ou analytics

3. API externes

ChatAurX utilise des API tierces pour générer les réponses IA. Vos prompts sont envoyés uniquement pour traitement et ne sont pas conservés par les fournisseurs.

4. Sécurité

Ne partagez jamais d'informations confidentielles.

L'IA peut générer des réponses inexactes. Ne vous fiez pas à ChatAurX pour des décisions critiques : médicales, juridiques ou financières.
