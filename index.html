Confidentialité & Utilisation

ChatAurX - Dernière mise à jour : Juin 2026

Confidentialité

1. Stockage des données

Toutes vos conversations sont stockées localement sur votre appareil. ChatAurX n'envoie aucune donnée vers des serveurs externes.

Vos messages, historiques et préférences restent dans le localStorage de votre navigateur. Vous gardez le contrôle total.

2. Aucune collecte

  • Nous ne collectons pas vos conversations
  • Nous ne vendons aucune donnée personnelle
  • Nous ne trackons pas votre activité
  • Aucun cookie tiers ou analytics

3. API externes

ChatAurX utilise des API tierces pour générer les réponses IA. Vos prompts sont envoyés uniquement pour traitement et ne sont pas conservés par les fournisseurs.

Les fournisseurs d'API respectent leurs propres politiques de confidentialité. Nous recommandons de ne jamais partager d'informations sensibles.

4. Sécurité

Ne partagez jamais :

  • Mots de passe ou identifiants
  • Numéros de carte bancaire
  • Informations médicales sensibles
  • Documents confidentiels

L'IA peut générer des réponses inexactes. Ne vous fiez pas à ChatAurX pour des décisions critiques : médicales, juridiques ou financières.

Comment utiliser ChatAurX

1. Démarrer une conversation

Tapez votre message dans la zone de saisie en bas et appuyez sur Envoyer ou la touche Entrée.

L'IA répond en temps réel. Vous pouvez poser des questions, demander de l'aide, générer du texte, du code, etc.

2. Historique des chats

Cliquez sur l'icône Historique en haut à gauche pour voir toutes vos conversations.

  • Créer un nouveau chat avec le bouton +
  • Reprendre une conversation existante
  • Supprimer l'historique complet si besoin

3. Paramètres

Accédez aux paramètres via l'icône en haut à droite :

  • Changer la taille de police : Petit / Moyen / Grand
  • Voir les infos de l'app
  • Gérer les données locales

4. Fonctionnalités

  • Markdown : formatage riche avec **gras**, *italique*, listes
  • Code : coloration syntaxique pour 20+ langages
  • Maths : formules LaTeX automatiques $$x^2 + y^2$$
  • Liens : URLs cliquables automatiquement
  • PWA : installable comme app mobile

5. Raccourcis

  • Entrée : envoyer le message
  • Shift + Entrée : retour à la ligne
  • Swipe : ouvrir/fermer l'historique sur mobile
Limites & Responsabilités

Ce que ChatAurX ne fait PAS

  • Donner des conseils médicaux, juridiques ou financiers fiables
  • Accéder à Internet en temps réel
  • Mémoriser vos infos entre les sessions
  • Garantir 100% de précision

Vous êtes responsable de l'usage que vous faites des réponses. Vérifiez toujours les informations critiques auprès de sources officielles.

Usage interdit

  • Générer du spam ou du contenu frauduleux
  • Harceler ou menacer des personnes
  • Créer du contenu illégal ou violent
  • Reverse engineering de l'app
